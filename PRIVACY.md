# Privacy Policy

## Overview

This application is an open-source Flutter application that integrates with Google Calendar using the official Google Calendar APIs. Access to Google Calendar data is performed **only with explicit user consent** via Google OAuth 2.0.

## Data Access and Usage

The application may access the following Google Calendar data, strictly for user-requested functionality:

* Read calendar events
* Create new calendar events
* Modify existing calendar events
* Delete calendar events

Calendar data is accessed **only while the app is in use** and **only to perform actions explicitly initiated by the user**.

## Data Storage

* The application **does not store calendar data on any external servers**.
* Calendar data is processed locally on the user’s device.
* OAuth access and refresh tokens are stored securely on the user’s device using platform-provided secure storage mechanisms.
* No calendar content is logged, shared, sold, or transmitted to third parties.

## Third-Party Services

The application uses the following third-party services:

* **Google OAuth 2.0** for authentication
* **Google Calendar API** for calendar access

These services are used in accordance with Google’s API Services User Data Policy.

## Data Sharing

The application does **not**:

* Share personal data with third parties
* Sell personal data
* Use calendar data for advertising or analytics

[https://myaccount.google.com/permissions](https://myaccount.google.com/permissions)
